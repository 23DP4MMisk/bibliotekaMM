# Online biblioteka ***MyLibrary***

 `MyLibrary ir bibliotēka, kas apvieno divas galvenās grāmatu kategorijas: akadēmiskās grāmatas un grāmatas atpūtai, piemēram, zinātniskā fantastika, klasika un daudzi citi žanri.`  
 ## Nereģistrēta un reģistrēta viesa funkcijas  
 Nereģistrēta lietotāja īespejas:  
  * Pārlūkot grāmatas  
  *  Lasīt grāmatu īso aprakstu  
  * Lasīt atsauksmes        
    
Reģistrēta lietotāja īespejas:  
Tas pats, kas nereģistrēts lietotājs, bet ar papildu funkcijām:  
* Pievienot grāmatas izlasei  
* Lejupielādēt grāmatas PDF formātā  
* Rakstīt atsauksmes par izlasītajām grāmatām  
  
`Atkarību instalēšana citā datorā`  
  
1. cp .env.example .env
2. composer install  
3. npm install  
4. php artisan key:generate  
5. Rediģēt .env failu
6. php artisan migrate  
  
` Projekta uzsākšana`  
1. 1 terminala php artisan serve  
2. 2 terminala npm run dev
